## Suggested First Targets (learning-friendly)

Good starting programs for first reps:
- **Uniswap v2 (educational)** — massively audited, great for learning patterns
- **Smaller DeFi lending protocols** — Euler-style, often have edge cases
- **New L2 bridge contracts** — fresh code, bridge logic is complex

Avoid until experienced:
- Uniswap v3 (battle-hardened)
- Compound v3 (heavily audited)
- Any protocol post-major-audit with no code changes
